# Product Roadmap (Non-binding)

This roadmap outlines likely next improvements for product credibility and operational maturity.

## Near-term (0-2 months)

- Add lightweight end-to-end tests for critical product paths
- Add frontend linting + formatting workflow in CI
- Improve accessibility pass (keyboard nav, ARIA labels, color contrast)
- Expand dashboard filtering and saved views
- Add seeded demo scenarios selectable by industry

## Mid-term (2-4 months)

- Add webhook signature validation and ingestion retry queue
- Expand analytics into cohort and team-level performance views
- Add RBAC policies backed by backend role claims
- Add API key management UI and scoped service keys
- Add SLA and anomaly alerts to system status

## Longer-term (4+ months)

- Multi-tenant backend isolation improvements
- SSO (Google/Microsoft/SAML) for enterprise customers
- Full audit export package (CSV/JSON)
- Advanced workflow automation builder with rule testing sandbox
- AI quality monitoring with prompt/version governance

## Product Principles

- Keep AI explainable and human-controllable
- Preserve compatibility of existing API contracts
- Optimize for clear business outcomes over feature bloat
- Prioritize reliability, observability, and trust
